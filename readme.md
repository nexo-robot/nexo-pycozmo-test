# Programme de teste de la librairy nexo-pycozmo

## Ligne de commande importante

### Clonage du dépôt : 

```bash
git clone --recursive https://github.com/nexo-robot/nexo-pycozmo-test.git
```

### Mise a jour de pip
```bash
python -m pip install --upgrade pip
```

### Installation des lib
```bash
pip install -r requirement.txt
```

### Installation de la version modifier de pycozmo
```bash
cd nexo-pycozmo/
python setup.py install
```

### Lancement du logiciel :
```bash
python3.14 main.py
```
